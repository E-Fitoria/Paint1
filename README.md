# Paint1
* En este paint fue desarrolado para la clase de Geometría Computacional, se pueden dibujar distintintas figuras geométricas, 
  cambiar sus colores, cortar, mover y borrar.

### Tecnologías usadas
* Lenguaje de programación __C__".
* IDE TurboC 2.0
* Librería graphics.h
